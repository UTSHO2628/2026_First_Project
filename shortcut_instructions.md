The `run_app.bat` script has been created. Now, to create a one-click shortcut on your desktop, please follow these simple steps on your Windows PC:

### How to Create Your Desktop Shortcut

1.  **Find the Batch File:**
    *   Open your File Explorer.
    *   Navigate to your project directory: `D:\habit_tracker`
    *   You will see a new file named `run_app.bat`.

2.  **Create the Shortcut:**
    *   Right-click on the `run_app.bat` file.
    *   From the context menu, select **Send to** > **Desktop (create shortcut)**.

    ![Send to Desktop](https://i.imgur.com/WJmB8c7.png)

3.  **Rename the Shortcut (Optional):**
    *   Go to your desktop.
    *   You will find a new shortcut, likely named `run_app.bat - Shortcut`.
    *   Right-click on this shortcut, select **Rename**, and give it a clean name like **"Habit Tracker"**.

That's it! Now, whenever you double-click the "Habit Tracker" icon on your desktop, the application will start instantly.
