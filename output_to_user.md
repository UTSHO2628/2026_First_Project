The Habit & Finance Tracker application has been successfully launched.

You should now see a desktop application window open.

**To interact with the application:**

1.  **Date Selection:** At the top, you'll see dropdowns for Year, Month, and Day. Select the desired date for data entry or viewing.
2.  **Data Entry:** Fill in the input fields for Health & Routine, Productivity, Social Media Activity, Finance, and Optional notes.
    *   **Validation:** Numeric fields will only accept non-negative numbers.
3.  **Save Data:** Click the "Save Data" button at the bottom right to store your entries for the selected date. Data is saved locally in the `data/` folder (one JSON file per year).
4.  **Show Analysis:** Click the "Show Analysis" button at the bottom left to open a new window.
    *   In the analysis window, select a Year and either a specific Month or "All" for a yearly summary.
    *   Click "Generate Report" to see monthly or yearly financial and habit summaries.

You can close the application window to exit.
